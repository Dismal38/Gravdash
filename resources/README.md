# GRAV-SHIFT Brand Assets

These are the master Play Store + Android assets, all generated procedurally
to match the in-game neon retro aesthetic exactly.

| File | Size | Purpose |
|---|---|---|
| `icon.png` | 1024×1024 | Master launcher icon (use for Play Store listing) |
| `icon-fg.png` | 1024×1024 | Foreground layer for Android adaptive icon (transparent BG) |
| `splash.png` | 2732×2732 | Capacitor splash, center-cropped to any device size |
| `feature.png` | 1024×500 | Play Store **Feature graphic** (required for listing) |

## Already wired into Android

The Android density variants (mdpi → xxxhdpi, portrait + landscape splash —
**123 files total**) have already been generated by `@capacitor/assets` and
copied into `/app/frontend/android/app/src/main/res/`. You don't need to
do this again unless you change the source PNGs.

## Re-generating

If you tweak the source PNGs:

```bash
cd /app/frontend
cp /app/resources/icon.png      assets/icon.png
cp /app/resources/icon-fg.png   assets/icon-foreground.png
cp /app/resources/splash.png    assets/splash.png
# icon-background.png is a solid #050508 1024² (already generated)
npx capacitor-assets generate --android --assetPath assets
```

## To regenerate the source PNGs themselves

```bash
python /app/scripts/gen_assets.py
```

This runs in ~2s using Pillow + FreeSans Bold.

## Play Store upload checklist

When uploading to the Play Console you'll need:

- **App icon**: `icon.png` (the Console will ask for 512×512 — Console
  auto-resizes from 1024).
- **Feature graphic**: `feature.png` (1024×500 — exact size required).
- **Phone screenshots** (min 2): take from a real device or emulator running
  the signed APK.

For the listing description, here's a copy-paste starter:
